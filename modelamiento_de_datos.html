MODELAMIENTO DE DATOS

Modelamiento de Datos

En en mundo del diseño de bases de datos, es crucial el hecho de que el analista de sistemas comprenda plenamente las necesidades de sistematización del usuario final.

Para lograr esta plena comprensión, como primer paso debe el usuario final expresarle sus necesidades al analista, en una fase que se ha denominado "elicitación" de requisitos o levantamiento de requerimientos.

Fuente. www.pngwing.com/es/search?q=requisito

Luego de hacer este proceso de "elicitación de requisitos, como segundo paso el analista debe construir un modelo de datos que dé cuenta de las necesidades de manejo de datos expresadas por el usuario final.

Este modelo de datos es una representación grafica de los datos que va a manejar el usuario en sus procesos y la relación que existen entre dichos datos.

El proceso de modelamiento de datos, el cual es el tema de este módulo, permite construir dicho modelo de datos y se constituye en una herramienta primordial para validar con el usuario final la comprensión del problema.

Fuente. http://elies.rediris.es/elies9/5-1.htm
