CONCEPTOS BASICOS

En este módulo exploraremos una serie de conceptos necesarios para entender todo el proceso de diseño de una base de datos.

Los conceptos que se abordarán responderán a preguntas tales como qué es una base de datos, cuál es la diferencia con un DBMS, cuáles son los niveles de abstracción de datos, qué es SQL, cuáles son los diferentes modelos de datos existentes, cuáles son los usuarios que utilizan una base de datos, incluido el DBA, qué es un backup y cuáles son los tipos de backups que existen, entre otras preguntas.

Estas preguntas son importantes responderlas para poder entender, de forma satisfactoria, el proceso de diseño de una base de datos.

Fuente. https://www.opertek.com/blog/que-es-iiot/
